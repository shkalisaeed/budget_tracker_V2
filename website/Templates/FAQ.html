FAQs & Documentation

All the data uploads are done through the profile page, they can all be accessed on the sidebar on that page.
Yes, The Budget Manager allows for importing of financial data from other sources. It supports file types such as .csv and .xlsx. This makes it easy to integrate your existing financial information into the platform.
Your personal and financial security is a top priority in The Budget Manager. The platform uses encryption technologies to safeguard your sensitive information, which is stored off-site to further enhance security.
Adding income or expenses in The Budget Manager is straightforward. To add an expense, go to "Profile", then on the left panel, select "Upload Individual Expense". For adding income, follow the same process but instead select "Income".
Yes, The Budget Manager provides an option for creating customized budgets. To create a new budget, select "Budget Creator" from the top panel, then click on "add new". Here, you can specify your budget details according to your needs.
Absolutely. To set financial goals, navigate to the top panel and select "Goals", then click on "add new". You can then specify your goal details, such as the amount and deadline.
